# 📘 Financial Analyzer

A simple, data‑driven personal finance tracking program built using Python.  
This project demonstrates the use of:

- Functions  
- Object-Oriented Programming (OOP)  
- Files (TXT)  
- Lists  
- Data processing & statistics  

---

## 📌 Features

### ✔ Add new expenses  
Users can input:
- Date  
- Category  
- Amount  

### ✔ View all expenses  
Displays all stored expense records.

### ✔ View statistics  
Automatically calculates:
- Total spending  
- Average spending  
- Highest expense  
- Lowest expense  

### ✔ Persistent storage  
All expenses are saved in **expenses.txt**  
Data loads automatically when the program starts.
